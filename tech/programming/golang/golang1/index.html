# 概述

Golang 的数据类型按存储方式分为两大类：
1. 值类型

  • int
  • float
  • bool
  • string
  • struct
  • 数组

2. 引用类型

  • 指针
  • slice
  • map
  • channel
  • interface

由于 channel 和 interface 涉及更加高级的编程模式，此处不做介绍。

# 值类型数据

值类型数据的特点：
（1）变量直接存储值。
（2）值类型的数据存储在栈内存空间中，栈在函数调用完内存会被释放。

# 指针类型数据

指针类型数据的特点：
（1）引用类型变量存储的是一个地址，这个地址存储数据的值。
（2）引用数据类型的数据存储在堆内存空间中，通过 GC 回收。
